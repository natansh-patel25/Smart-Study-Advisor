*AI-Based Smart Study Advisor*

OVERVIEW
The problem addressed in this project is helping students understand their academic performance and improve their study habits. This project uses basic machine learning techniques in Python to predict a student’s final marks based on inputs such as study hours, attendance, and previous marks. Along with prediction, the system also provides useful suggestions to help students improve their performance.

FEATURES
1. Predicts final marks based on user input.
2. Takes inputs like study hours, attendance percentage, and previous marks.
3. Provides personalized suggestions for improvement.
4. Simple and easy-to-use command line interface.
5. Demonstrates basic use of machine learning in a real-life scenario.

TECHNOLOGIES USED
1. Python
2. Pandas (for handling data)
3. Scikit-learn (for machine learning model)

STEPS TO INSTALL AND RUN
1. Make sure Python is installed on your system.
2. Open Command Prompt or Terminal.
3. Navigate to the project folder.
4. Install required libraries using:
   pip install pandas scikit-learn
5. Run the program using:
   python main.py

INSTRUCTIONS FOR TESTING
1. Run the program and enter study hours (e.g., 3 or 5).
2. Enter attendance percentage (e.g., 60 or 80).
3. Enter previous marks (e.g., 50 or 70).
4. The system will display predicted final marks.
5. Check the suggestions provided by the system.
6. Try different inputs to see how the prediction and suggestions change.

EXAMPLE
Input:
Study Hours: 2
Attendance: 60
Previous Marks: 50

Output:
Predicted Final Marks: 54.89

Suggestions:
- Try to increase your study hours
- Improve your attendance
- Revise basic concepts and practice more

CONCLUSION
This project shows how machine learning can be applied in a simple way to solve real-world problems. It helps students understand their preparation level and provides guidance for improvement.
